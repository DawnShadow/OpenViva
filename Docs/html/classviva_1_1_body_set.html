viva.BodySet Class Reference

Represents a collection of AnimationStates that share a similar body pose. Example: "walk forward", "standing idle", "standing waving hello" all belong to a BodySet named "standing" because they are all being performed while standing. This is used for pathfinding animation transitions to animate going from one BodySet to another in a contextually coherent way. Detailed Description

Represents a collection of AnimationStates that share a similar body pose. Example: "walk forward", "standing idle", "standing waving hello" all belong to a BodySet named "standing" because they are all being performed while standing. This is used for pathfinding animation transitions to animate going from one BodySet to another in a contextually coherent way. Example: Going from a "sleep face down" animation to a "standing jump" requires the character to roll over and stand up before jumping.
