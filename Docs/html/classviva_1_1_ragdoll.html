LimitGroup muscleLimit = new LimitGroup()
 The LimitGroup controlling the maximum muscle percent when applying animations on the ragdoll. Adding a limit of 0 percent will make the ragdoll collapse on the floor and have zero influence from an animation.
 
LimitGroup pinLimit = new LimitGroup()
 The LimitGroup controlling the maximum pin percent when applying animations on the ragdoll. Adding a limit of 0 percent will make the ragdoll collapse on the floor but retain their animation pose.

Detailed Description

The class used to represent 16-bone physics ragdolls. You can control the strength of each animated muscle or the ragdoll as a whole with muscleLimit and pinLimit.
