\doxysubsection{Detailed Description}
Class for handling a \mbox{\hyperlink{classviva_1_1_character_i_k}{Character\+IK}} handle. The most recent handle will be used to animate the IK while the rest will fade away and be removed. 

The following example makes the right arm of a character point towards the center of the map.

\DoxyCodeLine{character.biped.rightArmIK.AddRetargeting( delegate( out Vector3 target, out Vector3 pole, out Quaternion handRotation )\{}

\DoxyCodeLine{    pole = \textcolor{keyword}{new} Vector3( 0, 0, 0 );}
\DoxyCodeLine{}
\DoxyCodeLine{    handRotation = Quaternion.identity;}
\DoxyCodeLine{\}, out IKHandle activeHandle );}
\DoxyCodeLine{}
